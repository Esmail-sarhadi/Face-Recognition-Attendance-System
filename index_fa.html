سیستم حضور و غیاب با تشخیص چهره 📸👤

این پروژه یک سیستم حضور و غیاب نوآورانه با استفاده از تشخیص چهره است که از وب‌کم برای ثبت و شناسایی چهره‌ها در زمان واقعی استفاده می‌کند. سیستم چهره‌های ثبت شده را با چهره‌های شناخته شده مقایسه کرده و جزئیات حضور و غیاب (نام و زمان) را در فایل CSV ثبت می‌کند. این پروژه از کتابخانه face_recognition برای تشخیص و شناسایی چهره و از OpenCV برای پردازش ویدیویی استفاده می‌کند.

✨ ویژگی‌ها

  • تشخیص و شناسایی چهره در زمان واقعی: تصاویر ویدیویی را از وب‌کم گرفته و چهره‌ها را در زمان واقعی شناسایی می‌کند.
  • ثبت حضور و غیاب: جزئیات حضور و غیاب (نام و زمان) را در فایل CSV ثبت می‌کند.
  • مدیریت چهره‌های شناخته شده: به آسانی می‌توان چهره‌های شناخته شده را برای شناسایی اضافه و مدیریت کرد.
  • بازخورد تصویری: یک مستطیل و برچسب نام را دور چهره‌های شناسایی شده در تصویر ویدیویی نمایش می‌دهد.

🛠️ الزامات

  • پایتون ۳.۶+ 🐍
  • OpenCV 📸
  • face_recognition 👤
  • numpy 🧮

🚀 نصب

  1. کلون کردن مخزن 
    git clone https://github.com/your-username/face-recognition-attendance.git
cd face-recognition-attendance
  2. نصب بسته‌های مورد نیاز 
    pip install opencv-python face_recognition numpy
  3. تهیه تصاویر چهره‌های شناخته شده
    • تصاویر چهره‌های شناخته شده خود را در دایرکتوری photos قرار دهید.
    • اطمینان حاصل کنید که تصاویر با نام‌های مناسب (مانند john_doe.jpg) نام‌گذاری شده‌اند.
  4. اجرای برنامه 
    python main.py

📋 استفاده

پس از اجرای برنامه، فید وب‌کم نمایش داده می‌شود. سیستم چهره‌ها را در زمان واقعی شناسایی و تشخیص می‌دهد. اگر چهره‌ای شناسایی شود، نام در فید ویدیویی نمایش داده می‌شود و جزئیات حضور و غیاب در فایلی با نام attendance.csv ثبت خواهد شد.
